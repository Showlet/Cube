# Controls

|-------------------+---------------|
| Moving            | w / a / s / d |
| Delete block      | Left Click    |
| Add block         | Right Click   |
| Select a block    | Roullette     |
| Sneaking          | Ctrl          |
| Run               | Shift         |
| Jump              | space         |
| Fly / NoClip Mode | v             |
| Wireframe Mode    | y             |
| Display info      | f3            |
| Fullscreen Mode   | f10           |
| Open Map          | LShift-o      |
| Save              | LShift-w      |
| Create Random Map | LShift-r      |
| Creer Flat Map    | LShift-f      |
| Delete Cache      | LShift-f5     |
|-------------------+---------------|
